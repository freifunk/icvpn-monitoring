Freifunk ICVPN Monitoring About

Das Monitoring-System nutzt als Quelle für die konfigurierten BGP-Sessions folgendes GIT:

ICVPN-META auf GitHub

Die Schlüssel für die darunterliegende tinc-Verbindung werden dem folgenden GIT entnommen:

ICVPN auf GitHub

Die Aktualisierung der Konfiguration für die TINC-Verbindungen sowie der BGP-Sessions erfolgt 4 mal pro Tag. Falls das Monitoring veraltete Daten anzeigt aktualisiere bitte die Daten deiner Community in einem oder beiden oben genannten GIT Repos.

Auf dieser Seite werden die jeweils letzten Zeitpunkte dargestellt zu denen Prozesse des Monitorings abgelaufen sind. Falls zu irgendeinem Zeitpunkt keine aktuellen Daten auf diesen Seiten dargestellt werden, trotzdem die Daten im icvpn-meta GIT aktuell sind, ist das jeweilige Update eventuell noch nicht erfolgt.

Update tincd configvor 3 Stunden und 46 Minuten

Aktualisiert die in tincd konfigurierten Partnermit den Informationen des icvpn GIT.Repositories.

Update bird bgp peer configvor 3 Stunden und 31 Minuten

Generiert die Liste der BGP-Peers anhand der Daten aus dem icvpn-meta GIT. Bird baut die BGP-Sessions zu anderen Teilnehmern am ICVPN auf.
